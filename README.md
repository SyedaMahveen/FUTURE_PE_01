## AI Website Copy Generator for Local Businesses

## Project Overview

This project demonstrates the use of Prompt Engineering to generate professional, persuasive, and conversion-focused website copy for local businesses using Generative AI.

The prompt framework can generate complete website content, including hero sections, service pages, testimonials, FAQs, and contact information for different industries.

## Objective

To design a reusable prompt system capable of creating website-ready content for local businesses efficiently.

## Features

* AI-powered website copy generation
* Professional and persuasive content
* Industry-specific customization
* Strong call-to-action sections
* FAQ generation
* Website-ready outputs
* Reusable prompt framework
